# v2 대표 데모 시나리오

## 시나리오명
옥외 작업 + 강풍주의보 + 지게차/추락 위험

## 입력 예시
- 위치: 서울
- 작업 유형: 옥외 작업
- 작업 시간: 주간
- 위험 키워드: 추락, 지게차
- 사용 물질: 벤젠(선택 확장)
- 기상 신호: 강풍주의보 또는 호우/건조 특보

## 시스템이 보여줄 것
### 1. 위험 요약
- 오늘 작업의 핵심 위험 3개
- 기상으로 인해 추가되는 위험
- 작업중지/주의/가능 여부를 신중한 문구로 제시

### 2. 관련 근거
- 관련 법령
- KOSHA 가이드/사례
- 필요 시 MSDS 규제 정보

### 3. 현장 조치
- 즉시 필요한 조치
- 작업 전 확인 사항
- 관리자/반장 전달 포인트

### 4. 산출물
- 위험성평가 초안
- TBM 브리핑 문안
- TBM 일지 초안

## 산출물 예시 틀
### 위험성평가 초안
- 작업명
- 유해위험요인
- 위험도
- 감소대책
- 관련 근거

### TBM 브리핑
- 오늘 핵심 위험
- 반드시 지킬 조치
- 날씨 관련 추가 주의사항
- 1분 브리핑 문안

### TBM 일지
- 일시
- 작업내용
- 참석자
- 전달내용
- 확인사항
- 특이사항
- 후속조치

## 시연 포인트
- 단순 검색이 아니라 **실제 현장 산출물 생성**이 핵심
- 공공데이터가 법령/가이드/기상 신호로 동시에 쓰인다는 점 강조
- live API 일부 실패 시에도 fallback으로 흐름 유지 가능해야 함

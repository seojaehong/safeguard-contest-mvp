# SafeGuard 제출 직전 재점검 체크리스트

## 제품 메시지
- [x] 한 줄 소개를 `오늘 작업 설명 한 줄을 위험성평가·TBM·안전교육 기록으로 바꾸는 산업안전 코파일럿`으로 통일
- [x] 발표와 화면에서 `검색 서비스`보다 `현장 산출물 생성과 실행 지원` 메시지를 우선 사용
- [x] 외국인 근로자, 신규 투입자, 교육 연계는 확장성 포인트로 정리

## 데모 화면
- [x] 홈 화면에서 위험성평가, TBM 브리핑, TBM 일지, 안전교육 기록 초안이 한 흐름으로 보임
- [x] `오늘 팀에 다시 물어야 할 질문 3개` 노출
- [x] 근거/출처와 live, partial, mock, fallback 상태를 구분해 설명
- [x] 외국인 근로자용 쉬운 문장 브리핑, 다국어/TTS는 향후 확장으로 명시

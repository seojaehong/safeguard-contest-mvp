# SafeGuard Pricing Ladder

## 기준

이번 단계의 가격 사다리는 결제 기능이 아니라 제품 포지셔닝과 잠금 표시 기준입니다. 실제 결제, 구독, 기능 차단 로직은 후속 설계에서 다룹니다.

## Free: 1회 생성 체험

사용자 약속:

- 오늘 작업 한 줄로 안전 문서팩을 생성해 봅니다.
- 위험성평가표, TBM, 외국인 전송본의 가치를 확인합니다.

포함:

- 11종 문서팩 1회 생성.
- 외국인 10개 언어 전송본.
- PDF, Excel, HWPX 다운로드.
- 공식 근거 기반 보조자료 고지.

잠금 표시:

- 현장 이력 저장.
- 어제 대비 변경.
- 증빙 라이브러리.
- 승인/봉인/다현장 리포트.

## T1: 1인 1현장

제품 카피:

> 매일 같은 현장을 다시 열고, 어제와 달라진 작업만 입력하세요.

해제 가치:

- 현장 프로필 저장.
- DailyEntry 이력.
- 마지막 작성 표시.
- 어제 대비 변경 입력.
- 브라우저 임시 저장과 Supabase 저장의 차이 안내.

사용자 화면:

- `현장 저장 가능`
- `마지막 작성`
- `오늘 바뀐 점`
- `로그인하면 이력이 보관됩니다`

## T2: 1인 다현장

제품 카피:

> 여러 현장의 공통 필드를 한 번만 고치고, 제출용 문서 품질을 맞추세요.

해제 가치:

- SharedContext 편집.
- 영향받는 문서 하이라이트.
- EvidenceLibrary 기본 기능.
- Excel/HWPX 제출형 품질 강화.
- 보조 내보내기 형식 접기.

사용자 화면:

- `공통 필드 동기화`
- `문서 반영 영향`
- `증빙 추가`
- `제출형 출력`

## T3: 팀

제품 카피:

> 작성, 검토, 승인, 전파 이력을 한 흐름으로 남기세요.

해제 가치:

- AuditTrail.
- ApprovalChain.
- 봉인 PDF 초안.
- 외국인 회신 수집.
- 교육 확인 이력.

사용자 화면:

- `검토 요청`
- `승인 후 발행`
- `수정 이력`
- `회신 확인`

주의 문구:

- 봉인 PDF는 변조 확인을 돕는 감사 대응 기록입니다.
- 공식 법적 효력 판단은 별도 검토가 필요합니다.

## T4: 본사 EHS

제품 카피:

> 본사는 여러 현장의 위험 신호와 교육 확인 상태를 한 화면에서 봅니다.

해제 가치:

- MultiSiteDashboard.
- 월간 리포트.
- Webhook.
- 그룹웨어 연동.
- 현장 상태 요약.

사용자 화면:

- `다현장 현황`
- `월간 리포트`
- `외부 시스템 연동`
- `현장별 확인 필요`

## 구현 원칙

- 이번 문서화 단계에서는 가격별 UI 문구와 잠금 상태만 정의합니다.
- 기능 접근 차단, 결제 연동, 청구, 환불, 플랜 변경은 구현하지 않습니다.
- 잠긴 기능은 클릭 시 “파일럿 플랜에서 제공” 안내를 보여주는 수준으로 설계합니다.
- 가격은 제품 소개와 영업 검증용이며 코드 상수로 고정하지 않습니다.

## 발표에서 그대로 쓸 문장
- Law.go는 위험성평가와 TBM의 법령 근거 축입니다.
- 기상청 데이터는 초단기실황, 초단기예보, 단기예보를 묶어 작업 전 위험 신호를 즉시 조치와 TBM 브리핑으로 연결합니다.
- 고용24 데이터는 외국인 근로자와 신규 투입자 교육 연계에 사용합니다.
- KOSHA는 현재 공식 Guide·교육자료를 시나리오별로 매핑해 TBM과 위험성평가 문구를 보강하는 축입니다.

## 문서 일치 규칙
- 모든 문서에서 제품 정의는 `실행 문서 생성형 안전 코파일럿`으로 통일한다.
- 데이터 상태 표기는 `live`, `fallback`, `planned`만 사용한다.
- `검색 서비스`라는 표현은 쓰지 않는다.

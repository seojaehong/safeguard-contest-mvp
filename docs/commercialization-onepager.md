## 한 줄 정의

SafeClaw는 현장관리자가 오늘 작업을 한 줄로 입력하면 위험성평가표, 작업계획서, TBM일지, 안전보건교육 기록, 외국인 공지, 전파 이력을 한 작업공간에서 처리하는 안전 문서 운영 SaaS입니다.

## 상용 v1 범위

- 관리자 로그인 후 문서팩과 근로자 정보를 저장합니다.
- 문서팩 생성 결과를 다시 열 수 있는 이력 흐름을 제공합니다.
- 위험성평가표, 작업계획서, TBM, 교육일지의 구조화 XLSX 출력을 제공합니다.
- HWP와 PDF 출력은 제출 보조 초안으로 제공합니다.
- 이메일과 SMS를 제출 기준 정식 채널로 둡니다.
- 카카오와 밴드는 승인 전이라 제출 기준에서 제외합니다.
- 출력물은 공식자료 기반 초안이며, 현장 검토 후 사용하도록 고지합니다.

## 닫힌 운영 흐름

1. 작업 입력.
2. 법령, KOSHA, 재해사례, 기상, 교육 데이터 반영.
3. 위험성평가표, 작업계획서, 허가/점검표, TBM일지 생성.
4. 사용자가 AI 보완 제안을 편집 후 삽입.
5. PDF/HWP/XLSX 다운로드.
6. 메일/SMS 전파.
7. 관리자 이력에서 재열기.

## 왜 지금 쓸 수 있는가

- 법령, KOSHA, 기상, 재해사례, 내부 지식 DB를 문서 생성 근거로 연결합니다.
- AI 보완은 자동 덮어쓰기가 아니라 `제안 -> 편집 -> 삽입` 흐름입니다.
- 개인정보는 전파와 교육 확인에 필요한 최소 항목만 사용합니다.
- 발송 전 수신자, 채널, 언어, 메시지를 확인합니다.
- 최신 구조화 출력 smoke에서 위험성평가표, 작업계획서, TBM, 교육일지 다운로드 흐름이 확인됐습니다.

## 정직한 고지

SafeClaw는 공식자료 기반 초안과 현장 기록 보조 도구입니다. 최종 제출, 현장 조치, 회사 양식 반영은 현장 확인 절차를 거쳐 확정합니다.

- HWPX 원본 공공기관 셀 단위 복제는 현재 제출 기준에 포함하지 않습니다.
- PDF는 제출 보조용으로 사용하고, 공식 제출 전 화면과 파일을 확인합니다.
- 카카오와 밴드는 승인 완료 전까지 정식 채널이 아닙니다.
- production 저장과 전파는 제출 직전 같은 계정으로 한 번 더 재현합니다.

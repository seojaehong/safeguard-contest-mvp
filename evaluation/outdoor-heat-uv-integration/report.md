# 옥외작업 폭염·자외선 API 보강 리포트

## 적용 범위
- 기상청 생활기상지수 자외선 `getUVIdxV4` adapter 추가
- 기상청 생활기상지수 여름철 체감온도 `getSenTaIdxV4` adapter 추가
- 한국에너지기술연구원 실시간 홍반자외선 `UlvryService/getUlvryPredcInfo` adapter 추가
- 위험성평가표, 작업계획서, TBM, 안전보건교육 기록, 현장 공유 메시지에 한여름 옥외작업 문구 반영

## 공식 가이드 반영 요약
옥외작업자 건강보호 가이드(폭염)의 핵심은 폭염 정보 확인, 물·그늘·휴식, 민감군·신규자·중작업자 추가관리, 가장 더운 시간대 작업 조절, 동료 작업자 상호관찰, 열사병 등 이상 징후 발생 시 즉시 작업중단 및 응급조치입니다. SafeGuard는 이를 위험성평가의 유해·위험요인과 감소대책, TBM 질문, 안전교육 확인방법에 나누어 반영합니다.

## 로컬 Smoke 결과
- 기상청 초단기실황, 초단기예보, 단기예보, 기상특보 live 확인
- 기상청 생활기상지수 자외선 `getUVIdxV4` live 확인
- 기상청 생활기상지수 여름철 체감온도 `getSenTaIdxV4` live 확인
- 한국에너지기술연구원 실시간 홍반자외선 `getUlvryPredcInfo`는 필수 파라미터 반영 후 `NORMAL_SERVICE`까지 확인했습니다. 현재 시각 조건에서는 `totalCount 0`이라 데이터 없음으로 처리합니다
- `/api/ask`: 위험성평가/TBM/안전교육/공유메시지 모두 옥외 폭염·자외선 append 확인

## 제출 문구 권장
한여름 옥외작업 시 기상청 현재·예보·특보와 생활기상지수의 자외선·체감온도, 국가중점 데이터 후보인 실시간 홍반자외선 데이터를 결합해 폭염·자외선 노출 위험을 위험성평가, TBM, 안전보건교육 기록, 현장 공유 메시지에 자동 반영합니다.

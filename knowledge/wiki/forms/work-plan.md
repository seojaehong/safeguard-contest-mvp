# 작업계획서

## 필수 섹션

- 작업개요
- 작업순서
- 장비·인원·허가 확인
- 작업중지 기준
- 확인 근거

## 반영 근거

- [지게차의 안전작업계획서 작성지침](https://oshri.kosha.or.kr/kosha/intro/northernGyeonggiBranch_A.do?articleNo=351943&attachNo=199463&mode=download) - 지게차 작업계획, 운행경로, 신호방법, 접근금지구역 확인 항목을 보강한다.
- [화기작업 화재·폭발 예방 매뉴얼](https://kosha.or.kr/kosha/data/screening_e.do?articleNo=235017&attachNo=113021&mode=download) - 작업허가, 가연물 제거, 화재감시자, 소화설비 확인 문구를 보강한다.
- [법제처 국가법령정보 산업안전보건법](https://www.law.go.kr/법령/산업안전보건법) - 사업주와 근로자의 안전보건 조치 의무를 문서 근거 초안에 연결한다.

## 사용 원칙

- SafeGuard 산출물은 공식자료 기반 초안입니다.
- 현장관리자는 작업 조건, 작업자 구성, 원청 요구 서식을 확인한 뒤 사용합니다.
- 출력물에는 확인자와 서명란을 유지합니다.
